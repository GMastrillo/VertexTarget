The VertexTarget application is envisioned as a full-stack platform comprising a React frontend with Tailwind CSS and , a FastAPI backend, and a MongoDB database. The core objective is to manage VertexTarget's portfolio and testimonials. The product requires a comprehensive admin dashboard () with a responsive sidebar and navigation, a main dashboard displaying statistics, and full CRUD (Create, Read, Update, Delete) functionality for projects (Portfolio) and testimonials.
Specific features include:
- Portfolio Management: Complete CRUD with a modal for creating/editing projects, confirmation for deletion, real-time search/filters, image previews, and dynamic technology tagging.

- Backend APIs: Robust RESTful APIs for Portfolio and Testimonials, including endpoints for listing, creating, editing, and deleting. A seed data endpoint is also required.
